Welcome to the jschess project

jschess is a JavaScript library to show chess games, run chess games or whatever. It is fully object oriented. The only reason the background is this ugly yellow is for testing purposes...:)

This is alpha material at the moment, use at your own risk! API is subject to change! Some links:
